# onAuthFlowClose

This callback will be called after the auth flow modal is closed.

## Usage

```tsx
<SophonContextProvider
  settings={{
    events: {
      onAuthFlowClose: () => {
        console.log("onAuthFlowClose was called");
      },
    },
  }}
>
  {/* ... rest of your app ... */}
</SophonContextProvider>
```
