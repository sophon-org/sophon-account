# onUserProfileUpdate

This callback will be called when user profile information is updated.

## Usage

```tsx
<SophonContextProvider
  settings={{
    events: {
      onUserProfileUpdate: (user) => {
        console.log("onUserProfileUpdate was called", user);
      },
    },
  }}
>
  {/* ... rest of your app ... */}
</SophonContextProvider>
```
