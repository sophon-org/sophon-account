# onAuthSuccess

This callback will be called after the user successfully authenticates.

## Usage

```tsx
<SophonContextProvider
  settings={{
    events: {
      onAuthSuccess: (args) => {
        console.log("onAuthSuccess was called", args);
      },
    },
  }}
>
  {/* ... rest of your app ... */}
</SophonContextProvider>
```
