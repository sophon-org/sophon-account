# Security

Sophon Account SDK inherits its security considerations from Dynamic. For detailed information on the security measures and best practices implemented by Dynamic, please refer to their [Security Overview](https://docs.dynamic.xyz/security/overview).

By leveraging Dynamic's robust security infrastructure, Sophon Account SDK ensures that your Web3 applications are protected against common threats and vulnerabilities. This includes secure authentication methods, data encryption, and regular security audits to maintain the highest standards of security.
