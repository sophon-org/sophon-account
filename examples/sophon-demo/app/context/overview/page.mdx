# Overview

## Sophon Context Provider

The `SophonContextProvider` is a provider component provides the context for the rest of the components to work. It must wrap all Sophon related components of your application.

## Sophon Wagmi Connector

The `SophonWagmiConnnector` is responsible for helping your Sophon application to integrate with Wagmi out of the box.
